// Readme.txt

Environment

Arduino IDE 1.5.8 + MediaTek LinkIt SDK 1.1.11

Notice

1.Now, LinkIt ONE cannot be compatible with Arduino IDE 1.6.x. Please remove your 1.6.x and install 1.5.x instead.

2.On Windows computer, open Device Manager and disable this device Alcatel Single RNDIS Interface. Restart the computer.

3.Don't forget to select Board LinkIt ONE and Port modem port.
