// Readme.txt

Environment

Microsoft Windows 7 with SP1 (64-bit Edition) + Arduino IDE 1.5.8 + MediaTek LinkIt SDK 1.1.11


Default Settings

Normal Bootup + USB Power Source + SPI


Notice

1.Now, LinkIt ONE cannot be compatible with Arduino IDE 1.6.x. Please remove your 1.6.x and install 1.5.x instead.

2.Open Device Manager and disable this device "Alcatel Single RNDIS Interface". Restart the computer.

3.Don't forget to select Board "LinkIt ONE" and Port "modem port".


Note

The board is changed to LinkIt Smart 7688 on October 21, 2015.
